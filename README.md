# 🌍 Tourism Package Purchase Prediction

A Machine Learning project that predicts whether a customer will purchase a tourism package using demographic, travel behavior, and marketing interaction data.

This project combines **EDA, Machine Learning, and an interactive Streamlit dashboard**.

---

## 📊 Project Overview

- Performed **Exploratory Data Analysis (EDA)** to understand customer behavior
- Built multiple ML models to predict package purchase
- Selected **Random Forest (best performing model)**
- Developed an **interactive Streamlit dashboard** for analysis and prediction

---

## 🏆 Model Performance

| Metric | Score |
|------|------|
Accuracy | **88.7%**
Precision | **85.6%**
Recall | **48.3%**
F1 Score | **0.62**
